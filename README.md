# Bangladesh 2.0

A modern and responsive website representing the vision of **Bangladesh 2.0**.  
This project highlights innovation, sustainability, infrastructure, empowerment, and the future development goals of Bangladesh.

---

## [🚀 Live Demo](https://amitchandradas2004.github.io/Bangladesh-2.0-Tailwind/) 

---

## 📌 Project Overview

Bangladesh 2.0 is a clean and modern landing page designed to showcase:

- 🌱 Innovation & Sustainability  
- 🏗 Infrastructure Development  
- 💡 Digital Transformation  
- ⚡ Green Energy  
- 👥 Empowerment  
- 📰 Recent News Section  
- 💳 Donation System UI  
- 📩 Newsletter Subscription  

The design focuses on simplicity, readability, and responsive layout for all devices.
